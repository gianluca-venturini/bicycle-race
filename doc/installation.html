Installation

How to run the application

Prerequisites

In order to install and run the application you need a Browser compatible with CSS3 and D3 javascript library, I suggest the last Chrome version.

You also need to enable javascript in your browser, if you don't know how see this guide.

You need a web server. If you don't have one don't worry, I'll explain how to use a basic Python web server.

You need a Database server that support MySQL, you can download and setup it following one of the following tutorial:

  • Windows: http://dev.mysql.com
  • Linux: http://www.yolinux.com
  • Mac osx: http://dev.mysql.com

Download the code

In oreder to download the code you can clone the GitHub repository with the command

git clone https://github.com/gianluca-venturini/bicycle-race.git

If you don't want to use git you can download the bicycle-race.zip and decompress it in a directory of your choice.

Start the MySQL database server

This step depends on your environment. I'll describe the procedure for starting the server under Linux environment since is the most common one. For Windows please go here, for Max OSX please go here. Linux command: 

/etc/init.d/mysql start

The first time that you start the server you need to create the database using the dump present into the repository. In order to create the DB and the user please refer to the manual. Now you need to download the dump from this address. There are different ways to insert the dump inside the database, this step depends on your environment:

  • Windows: download Heidisql and follow the manual on that site.
  • Linux: using the MySQL client you can import the database dump with 
    mysql -u username -p -h localhost DATA-BASE-NAME < data.sql
  • Mac osx: download Sequel Pro and import the dump as described in the manual.

Configuring the php files

In order to interact with the DB the PHP backend need to be configured. The application can be easily configured, you only need to open the file data/private/mysqli.int and insert your own database, username and password: 

$DB_server = "localhost";
$DB_username = "username";
$DB_password = "password";
$DB_database = "db_name";

Start the web server

In this section I explain how to start a basic Python based web server that allows you to see the project in local. If you want to use another webserver please refere to its manual for the setup.

In order to start the Python web server you need Python 2.7 or higher already installed in your system. If you don't have it please refer to this site.

In order to start the web server (in this example on port 8000) use this command:

python -m SimpleHTTPServer

If the server give you an output similar to that everything works fine:

Serving HTTP on 0.0.0.0 port 8000 ...

Now you can access the application through your browser at this URL: http://127.0.0.1:8000/
