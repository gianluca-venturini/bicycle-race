The source code

Where to find it

GitHub

The source code of this application is publicly avaliable on GitHub at this address: https://github.com/gianluca-venturini/bicycle-race

The easiest way to obtain the code is clone the entire repository with the command:

git clone https://github.com/gianluca-venturini/bicycle-race.git

ZipFile

You can also download the project as a single zip file: bicycle-race.zip
