Bicycle Race

Interactive Geographic Data Visualization Tool

Project description

The application consists in an interactive map of Chicago, with markers which represent the Divvy bikes stations, multiple controls and several charts and visualizations.
Its main purpose is to help the user investigate how the Divvy bikes are used, look for general patterns, and see how particular events affect those patterns.
